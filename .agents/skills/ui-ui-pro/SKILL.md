# UI/UX Skill

When building web applications, you MUST adhere to the following design principles:
1. **Rich Aesthetics**: Use modern web design best practices (vibrant color palettes, sleek dark modes, glassmorphism, and dynamic animations). Avoid generic colors.
2. **Dynamic Design**: Ensure the interface feels responsive and alive. Implement hover effects, interactive elements, and subtle micro-animations to enhance user engagement.
3. **Typography**: Use modern typography from Google Fonts (e.g., Inter, Roboto, or Outfit). Do not use browser defaults.
4. **Premium Feel**: Avoid creating simple minimum viable products. The design should feel premium and state-of-the-art.
5. **No Placeholders**: Use the `generate_image` tool to create working demonstrations and high-quality assets instead of relying on placeholders.
6. **Accessibility and Responsiveness**: Ensure all designs are fully responsive across devices and follow basic accessibility guidelines.
